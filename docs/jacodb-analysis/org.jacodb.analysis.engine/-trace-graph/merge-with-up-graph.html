fun mergeWithUpGraph(upGraph: TraceGraph, entryPoints: Set<IfdsVertex>): TraceGraph

Merges two graphs.

sink will be chosen from receiver, and edges from both graphs will be merged. Also, all edges from upGraph's sink to entryPoints will be added (these are edges "connecting" upGraph with receiver).

Informally, this method extends receiver's traces from one side using upGraph.
