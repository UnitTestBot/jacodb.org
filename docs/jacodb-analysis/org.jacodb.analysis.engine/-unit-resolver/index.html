UnitResolver

fun interface UnitResolver<UnitType>

Sets a mapping from JcMethod to abstract domain UnitType.

Therefore, it splits all methods into units, containing one or more method each (unit is a set of methods with same value of UnitType returned by resolve).

To get more info about how it is used in analysis, see runAnalysis.
