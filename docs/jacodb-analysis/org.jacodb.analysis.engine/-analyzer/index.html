Analyzer

interface Analyzer

Analyzer interface describes how facts are propagated and how AnalysisDependentEvents are produced by these facts during the run of tabulation algorithm by BaseIfdsUnitRunner.

Note that methods and properties of this interface may be accessed concurrently from different threads, so the implementations should be thread-safe.
